Overview

What this project is about

The project studies parallel BVH construction and efficient BVH traversal across sequential CPU, OpenMP multi-core CPU, and CUDA GPU implementations, using ray tracing scenes of increasing complexity to analyze build time, traversal time, load balancing, and memory behavior.

Core Focus

BVH first

Bounding Volume Hierarchies are the main systems problem. Ray tracing is there to evaluate how good the structure and traversal actually are.

Parallel Models

CPU and GPU

We compare sequential CPU, OpenMP multi-core CPU, and CUDA GPU implementations to study how different execution models handle irregular workloads.

Main Questions

Speed, scaling, behavior

We care about BVH construction, traversal efficiency, divergence, memory access patterns, and the tradeoffs between build cost and overall rendering performance.
